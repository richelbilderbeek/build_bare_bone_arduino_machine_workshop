    Build a bare-bone Arduino machine workshop¶

    Goal¶

    The goal of this workshop is to be able to create a bare-bone Arduino machine, i.e. a machine that only uses the ATmega328P chip of an Arduino (and not a complete Arduino Uno):

    The machine we'll build is a Minimal Pi Clock.

    Using the skills taught in this workshop, you can build you own machines using only the ATmega328P chip.

    Optionally, you can buy the tested components and make this into a proper Minimal Pi Clock.

    A Minimal Pi Clock. The electronics are soldered on a PCB and hidden inside the spherical casing.
